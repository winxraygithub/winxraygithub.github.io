白鲸科学上网的安装与配置指南

Windows系统安装教程

  1. 下载客户端
    访问白鲸官方网站，选择Windows版本进行下载。确保从官方渠道获取安装包，以避免恶意软件风险。
  2. 安装软件
    双击下载的安装文件，按照向导提示完成安装。建议选择默认安装路径，以避免权限问题。
  3. 注册与登录
    安装完成后启动客户端，使用邮箱或手机号注册账号，并完成登录。
  4. 选择服务器节点
    在服务器列表中选择一个合适的节点（通常优先选择地理位置较近或负载较低的节点），点击“连接”即可开始科学上网。

Mac系统安装教程

  1. 下载Mac版客户端
    从白鲸官网下载适用于macOS的安装包（通常为.dmg格式）。
  2. 安装应用
    打开下载的.dmg文件，将白鲸图标拖拽至“Applications”文件夹完成安装。
  3. 首次运行设置
    首次启动时，系统可能会提示“未识别的开发者”，需在“系统偏好设置→安全性与隐私”中手动允许运行。
  4. 连接服务器
    登录账号后，从节点列表中选择最优服务器，点击连接即可。

移动端安装指南（Android/iOS）

Android设备

  1. 获取APK文件
    由于Google Play可能不提供此类应用，建议从白鲸官网直接下载APK安装包。
  2. 允许未知来源安装
    进入“设置→安全”，开启“允许安装未知来源应用”选项，然后安装APK。
  3. 登录与连接
    打开应用后登录账号，选择节点并连接。

iOS设备

  1. App Store下载
    在App Store中搜索“白鲸VPN”并下载（若不可用，可能需要切换至其他地区商店）。
  2. 信任企业证书
    首次使用时，需在“设置→通用→设备管理”中信任开发者证书。
  3. 一键连接
    启动应用后登录，选择服务器并点击连接按钮。

  • 专用服务器：部分付费套餐提供流媒体或游戏专用服务器，适合有特殊需求的用户。

安全增强设置

  1. 启用防火墙功能
    在白鲸设置中开启内置防火墙，阻止恶意流量。
  2. 防止DNS泄露
    在“高级设置”中勾选“DNS泄露保护”选项，确保所有DNS查询均通过加密通道进行。

    开启“网络波动自动切换节点”功能，避免因网络不稳定导致连接中断。

2. 为什么有时连接速度较慢？如何解决？

可能原因及解决方案：
- 网络拥塞：尝试切换至负载较低的节点。
- 本地网络问题：重启路由器或切换网络环境（如从WiFi改为4G/5G）。
- 协议选择不当：在设置中尝试更换连接协议（如从UDP改为TCP）。

3. 使用白鲸是否合法？

白鲸作为技术工具，其合法性取决于使用目的和当地法律。
- 合规用途：学术研究、跨境商务、访问公开学术资料等通常被允许。
- 风险行为：绕过版权限制、访问非法内容等可能违反法律。
建议用户严格遵守所在国家/地区的相关规定。

4. 如何检测白鲸是否正常工作？

验证方法：
1. 访问测试网站：尝试打开Google、YouTube等通常受限的站点。
2. IP检测：访问whatismyipaddress.com，确认显示的IP地址与所选服务器位置一致。
3. DNS泄露测试：使用dnsleaktest.com检测是否存在DNS泄露。

5. 企业团队如何使用白鲸？

白鲸提供企业版解决方案，支持：
- 统一管理多账号
- 定制专属服务器
- 流量审计与日志记录
适合需要安全跨境协作的团队使用。
